Basic Terminal Commands


pwd -- print working directory
ls  -- short listing
cd  -- change directory

~   -- user home directory
/   -- root directory

mkdir -- create a directory
touch -- create a file

Note: After creating a file with touch command, like touch sagar.txt. If you try to run same command(touch sagar.txt), it will just update the timestamp.

ls -l -- long listing all the files
ls -a -- shows all the files (hidden too)

xrandr -o normal -- screen rotate to normal
